# Miscellaneous examples and tutorials

This directory contains a few examples and tutorials on some secondary
aspects of OpenPathSampling.

* `committors.ipynb`: How to use OPS for committor analysis, or to analyze the
  shooting points of path sampling simulation.
* `sequential_ensembles.ipynb`: How to build custom sequential ensembles
* `tutorial_storage.ipynb`: advanced storage features
* `tutorial_visualization.ipynb`: advanced features in the visualization of
  move "trees" and move schemes
